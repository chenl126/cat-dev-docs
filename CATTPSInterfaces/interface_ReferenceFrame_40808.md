# ReferenceFrame (Object)

**_Interface designed to manage reference frame associated to a TPS._**
Reference frame is composed of three boxes.

```VBScript
                        Reference Frame
                       /
                 _____|_____
                /           \
        ---------------------
        |   |   |Box|Box|Box|
        |   |   | 1 | 2 | 3 |
        ---------------------

```VBScript
